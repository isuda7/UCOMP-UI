설계규칙

System

문서구조

페이지 .page
페이지 머리말 .page-header
페이지 본문 .page-body
페이지 바닥글 .page-footer

영역규칙

전체 영영 -wrap
배치 영역 -area
섹션 영역 -sec

모듈규칙

모듈공통 [컴포넌트] [표현적 구조]
모듈공통 [컨텐츠] [내용적 성격]-[표현적 구조]
모듈구분 module-*
옵션구분 [강조], [모양], [방향], [정렬], [가변] 등
특성구분 is-[상태], has-[확장요소], type-[명사]

예외규칙

크기대응
  • 단위 클래스 (예).xs-1, .xl-1
  • 수치 클래스 (예).h-20
  • 컨텐츠 모듈로 컴포넌트의 크기를 제어 (예) .info-board .bul
유틸리티 [속성]-[값]
  • 기본 가이드에 포함하지 않으며 유연한 대응을 목적으로 필요시 사용한다.
  • 반복적인 네이밍을 [prefix]- 로 적용한다.
